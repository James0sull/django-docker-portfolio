\# Containerized Django Application



A foundational full-stack web application setup built with Django and isolated seamlessly using Docker and Docker Compose.



\## 🚀 Getting Started



Follow these steps to get the development environment running locally.



\### Prerequisites

Make sure you have \[Docker Desktop](https://www.docker.com/products/docker-desktop/) installed on your machine.



\### Installation \& Setup

1\. Clone this repository to your local machine.

2\. Build and start the Docker container:

&#x20;  ```bash

&#x20;  docker compose up --build
